# Django E-Commerce Analytics

A full-stack E-Commerce Analytics Platform built using Django and Django REST Framework.

## Features
- User Authentication
- Product Management
- Shopping Cart
- Secure Stripe Payment Integration
- RESTful APIs
- Real-Time Analytics Dashboard
- Sales, Orders, and Revenue Reports

## Tech Stack
- Python
- Django
- Django REST Framework
- PostgreSQL
- HTML
- CSS
- JavaScript
- Bootstrap
- Chart.js
- Stripe API
- Git & GitHub

## Getting Started

```bash
git clone https://github.com/purushotham018/Django-E-Commerce-Analytics.git
cd Django-E-Commerce-Analytics
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
